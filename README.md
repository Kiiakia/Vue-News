# Vue-News
该项目使用 vue3 + nodejs + express 建立一个包含了前后台的全栈项目。
## 项目说明
admin文件夹 —— 后台管理展示界面

server文件夹 —— 使用nodejs所写的后台api

web文件夹 —— 前台展示界面

## 项目使用
当前文件夹：

`npm i`

服务端启动

`cd server`

`npm i`

`npm start`

展示界面启动

`cd admin` or  `cd web`

`npm i`

`npm run serve`
